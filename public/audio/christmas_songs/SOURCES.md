# Royalty-Free Christmas Song Sources

Here are specific sources where you can find royalty-free instrumental Christmas songs:

## Recommended Sources

### 1. **Freesound.org**
- URL: https://freesound.org
- Search terms: "christmas instrumental", "jingle bells instrumental", "silent night instrumental"
- License: Check individual files (CC0, CC BY, etc.)
- How to download:
  1. Search for the song
  2. Filter by "Instrumental" and "Music"
  3. Download the file
  4. Rename to match the required filename

### 2. **YouTube Audio Library**
- URL: https://studio.youtube.com/channel/UC.../music
- Access: Requires YouTube account
- License: Free to use (check terms)
- How to download:
  1. Go to YouTube Studio > Audio Library
  2. Search for Christmas songs
  3. Download instrumental versions
  4. Extract 5-10 second snippets

### 3. **Pixabay Music**
- URL: https://pixabay.com/music/
- Search: "christmas", "holiday", "winter"
- License: Free for commercial use (Pixabay License)
- How to download:
  1. Search for Christmas music
  2. Filter by "Instrumental"
  3. Download and extract snippets

### 4. **Incompetech (Kevin MacLeod)**
- URL: https://incompetech.com/music/royalty-free/
- License: Check individual tracks (many are CC BY)
- How to download:
  1. Browse Christmas/holiday section
  2. Download instrumental tracks
  3. Extract recognizable snippets

### 5. **OpenGameArt.org**
- URL: https://opengameart.org
- Search: "christmas music", "holiday music"
- License: Various (check per file)
- Good for: Game-related music

## Alternative: Create Your Own

### Using Audacity (Free)
1. Download Audacity: https://www.audacityteam.org/
2. Record or import instrumental Christmas songs
3. Select a 5-10 second recognizable snippet
4. Export as MP3 (128-192 kbps)
5. Normalize volume

### Using AI Music Generators
- Suno.ai - Can generate instrumental snippets
- Udio - Can create short music clips
- Note: Ensure output is recognizable as the intended song

## Quick Download Script

Once you have URLs, you can:
1. Edit `scripts/download-audio-files.mjs`
2. Add URLs to the URL_MAP object
3. Run: `node scripts/download-audio-files.mjs`

## Tips

1. **Start with traditional carols** - These are often public domain
2. **Look for "karaoke" or "backing track" versions** - These are usually instrumental
3. **Check license carefully** - Even "free" music may have restrictions
4. **Test snippets** - Make sure 5-10 seconds is recognizable
5. **Normalize volume** - Keep all files at similar volume levels
